Tema 1

Estadística descriptiva

    La estadística descriptiva se encarga de resumir y exponer los datos de manera numérica y visual, utilizando herramientas como medidas estadísticas (media, mediana, moda, desviación estándar) y representaciones gráficas (histogramas, diagramas de dispersión). Su finalidad es ofrecer una descripción clara y concisa de las principales características de un conjunto de datos.

Definiciones

    1.1 Conceptos básicos

  • Teoría de decisión: La teoría de decisión investiga el proceso de selección entre opciones disponibles. Se divide en dos enfoques: el descriptivo, que examina decisiones reales, y el prescriptivo, que busca decisiones racionales empleando lógica y estadística.

  • Población: La población en estadística se refiere al grupo de individuos u objetos que comparten características similares y están siendo estudiados, abarcando todos los elementos sobre los cuales se buscan conclusiones estadísticas.

  • Muestra aleatoria: Muestra aleatoria: es una muestra obtenida de una población de unidades donde cada elemento tiene igual probabilidad de ser elegido, y la elección de un elemento no afecta la selección de los otros. Esto asegura que todos los elementos se seleccionan de forma independiente y equitativa.

  • Parámetros aleatorios: son variables que pueden tomar diversos valores, cada uno con una probabilidad determinada. La distribución de una variable aleatoria es una representación, ya sea en forma de tabla, gráfico o fórmula matemática, que muestra las probabilidades asociadas a cada valor posible de la variable.

    1.2 Descripción de datos:

  • Datos agrupados: Son los datos presentados en su forma original, mostrando cada valor individual tal como fue recopilado, sin ningún tipo de resumen o clasificación.

  • Datos no agrupados: Son los datos organizados en intervalos o categorías, donde los valores individuales se reúnen para facilitar el análisis, permitiendo identificar tendencias y patrones más fácilmente.

  • Frecuencia de clase: Cantidad de veces que los datos caen dentro de un intervalo específico.

  • Punto medio: Valores que definen los extremos de un intervalo.

  • Límites: Valores que definen los extremos de un intervalo.

    1.3 Medidas de tendencia central:

  • Media aritmética: La media aritmética es ampliamente utilizada como uno de los promedios más comunes. Se obtiene sumando todos los valores de un conjunto de datos y dividiendo la suma por la cantidad total de valores, destacándose por su utilidad estadística debido a sus propiedades.

  • geométrica y ponderada: La media geométrica ponderada es una versión de la media geométrica donde cada valor tiene un peso específico que influye en su contribución al promedio general.

  • Mediana: Es el valor central en un conjunto de datos cuando están ordenados. Si el conjunto tiene un número impar de elementos, la mediana es el valor exacto en la posición central.

  • Moda: La moda en estadística es el valor más frecuente en un conjunto de datos, importante junto con la media y la mediana para describir cómo están distribuidos los datos.

  • Medidas de dispersión: Son herramientas que muestran cómo los datos se distribuyen alrededor de su punto central, como la media o la mediana.

  • Varianza: Es una medida que indica cuánto se dispersan los datos respecto a su media. Se obtiene calculando la media de los cuadrados de las diferencias entre cada dato y la media del conjunto.

  • Desviación estándar: Es una medida que indica cuánto se dispersan los datos respecto a su media. Se obtiene calculando la media de los cuadrados de las diferencias entre cada dato y la media del conjunto.


  • Desviación media: Es la media de las diferencias absolutas entre cada dato y la media del conjunto. Ofrece una medida de dispersión que considera todas las observaciones de manera equitativa.


  • Desviación mediana: Es la mediana de las diferencias absolutas entre cada dato y la mediana del conjunto. Al igual que la desviación media, proporciona una medida de dispersión robusta ante valores extremos.


  • Rango: Es la diferencia entre el valor máximo y el valor mínimo en un conjunto de datos.

    1.4 Parámetros para datos agrupados:

  • Parámetros para datos agrupados: Los parámetros estadisticos como la media, la mediana y la desviación estándar se calculan con fórmulas específicas adaptadas a intervalos definidos en los datos agrupados.

    1.5 Parámetros para datos agrupados:

  • Parámetros para datos agrupados: Organiza los datos en intervalos o clases para mostrar cuántas observaciones caen en cada intervalo, revelando la frecuencia con la que aparecen diferentes valores.

    1.6 Técnicas de agrupación de datos:

  • Técnicas de agrupación de datos: Aseguran una organización efectiva de los datos en intervalos óptimos, determinando límites y distribución adecuada.

    1.7 Técnicas de muestreo

  • Técnicas de muestreo: Incluyen métodos como el aleatorio simple y el estratificado para seleccionar muestras representativas de una población más amplia.

    1.8 Histograma:

  • Histogramas: Gráficos que representan visualmente la distribución de frecuencias de datos agrupados, mostrando la cantidad de datos en cada intervalo.
