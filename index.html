Estadística descriptiva

    La estadística descriptiva es una parte de la estadística que describe los datos de forma numérica y gráfica. Para ello se utilizan datos estadísticos (media, mediana, moda, desviación estándar, rango, etc.) Esto con el fin de aclarar la compresión de los datos recopilados.

1.1 Conceptos básicos de estadística:

  • Teoría de decisión: La teoría de la decisión se centra en evaluar las diferentes opciones disponibles y elegir la opción que maximiza o minimiza, teniendo en cuenta la incertidumbre asociada a la información recopilada.
  • Población: En estadística, población se refiere a un grupo de personas o cosas que tienen características similares e incluye todos los elementos estudiados, para los cuales se buscan resultados.
  • Muestra aleatoria: La muestra aleatoria es una muestra obtenida de una población de unidades donde cada elemento tiene igual probabilidad de ser elegido, y la elección de un elemento no afecta la selección de los otros. Esto asegura que todos los elementos se seleccionan de forma independiente y equitativa..
  • Parámetros aleatorios: Son variables que pueden tomar diversos valores, cada uno con una probabilidad determinada. La distribución de una variable aleatoria es una representación, ya sea en forma de tabla, gráfico o fórmula matemática, que muestra las probabilidades asociadas a cada valor posible de la variable.

1.2 Descripción de datos:

  • Datos agrupados: Los datos agrupados son datos estructurados en intervalos. Dicho de otra forma, son datos que se juntan en intervalos para poder estudiarlos de manera conjunta.
  • Datos no agrupados: Son aquellos datos que se analizan de manera individual, sin agruparse en intervalos. Dicho de otra forma, cada valor del conjunto de datos se estudia por separado.
  • Frecuencia de clase: La frecuencia es la cantidad de datos que se encuentran en un intervalo particular, expresada como un número o un porcentaje.
  • Frecuencias Relativa: La frecuencia relativa indica cuántas veces aparece un valor en relación al total de datos de una muestra.
  • Punto medio: El punto medio es el valor que se encuentra en el centro o promedio de un conjunto de datos
  • Límites:Los límites en estadística son los valores que establecen los extremos de un intervalo particular.

    1.3 Medidas de tendencia central:

  • Media aritmética: Se obtiene sumando todos los valores de un conjunto de datos y dividiendo la suma por la cantidad total de valores, destacándose por su utilidad estadística debido a sus propiedades.
  • Geométrica y ponderada: La media geométrica ponderada es una versión de la media geométrica donde cada valor tiene un peso específico que influye en su contribución al promedio general.
  • Mediana: Es el valor central en un conjunto de datos cuando están ordenados. Si el conjunto tiene un número impar de elementos, la mediana es el valor exacto en la posición central.
  • Moda: La moda en estadística es el valor más frecuente en un conjunto de datos, importante junto con la media y la mediana para describir cómo están distribuidos los datos.
  • Medidas de dispersión: Son herramientas que muestran cómo los datos se distribuyen alrededor de su punto central, como la media o la mediana.
  • Varianza: Es una medida que indica cuánto se dispersan los datos respecto a su media. Se obtiene calculando la media de los cuadrados de las diferencias entre cada dato y la media del conjunto
  • Desviación estándar: Es una medida que indica cuánto se dispersan los datos respecto a su media. Se obtiene calculando la media de los cuadrados de las diferencias entre cada dato y la media del conjunto.
  • Desviación media: Es la media de las diferencias absolutas entre cada dato y la media del conjunto. Ofrece una medida de dispersión que considera todas las observaciones de manera equitativa.
  • Desviación mediana: Es la mediana de las diferencias absolutas entre cada dato y la mediana del conjunto. Al igual que la desviación media, proporciona una medida de dispersión robusta ante valores extremos.
  • Rango: Es la diferencia entre el valor máximo y el valor mínimo en un conjunto de datos.

    1.4 Parámetros para datos agrupados:

  • Los parámetros estadísticos como la media, la mediana y la desviación estándar se determinan utilizando fórmulas adaptadas a intervalos específicos dentro de la distribución de datos agrupados, lo que ayuda a interpretar de manera más clara la información contenida en estos conjuntos

    1.5 Distribución de frecuencias:

  • Las distribuciones de frecuencias son tablas que clasifican las categorías de una variable en filas y muestran las frecuencias de cada categoría en columnas. Ayudan a entender y analizar los datos de manera más sencilla.

    1.6 Técnicas de agrupación de datos:

  • Las técnicas de agrupación de datos en estadística son métodos que permiten organizar conjuntos de datos en grupos según sus similitudes. Estas técnicas investigan la estructura interna de los datos, descubren patrones y facilitan su análisis.

    1.7 Técnicas de muestreo:

  • Las técnicas de muestreo estadístico, como el muestreo aleatorio simple y el estratificado, son métodos empleados para elegir muestras representativas de una población más grande. Estas técnicas posibilitan inferir conclusiones sobre toda la población a partir del análisis de la muestra seleccionada.

    1.8 Histograma:

  • Un histograma es un gráfico que muestra la distribución de datos mediante barras, donde la altura de cada barra indica la frecuencia de los datos dentro de intervalos definidos.

    2.1 Técnicas de Conteo:

  • Las técnicas de conteo son métodos matemáticos que permiten determinar el número total de posibles combinaciones, permutaciones o arreglos de elementos en un conjunto.
  • 2.1.1 Principio aditivo: Este principio establece que si se tienen dos eventos mutuamente excluyentes (es decir, que no pueden ocurrir al mismo tiempo), el número total de maneras en que puede ocurrir uno u otro de los eventos es la suma de las maneras en que puede ocurrir cada evento por separado.
  • 2.1.2 Principio multiplicativo: Este principio afirma que si un evento puede ocurrir de m maneras y, después de que este evento ocurre, un segundo evento puede ocurrir de n maneras, entonces el número total de formas en que ambos eventos pueden suceder consecutivamente es el producto de m y n.
  • 2.1.3 Notación Factorial: El factorial de un número entero no negativo n, denotado como n!, es el producto de todos los enteros positivos desde 1 hasta n. Es una operación fundamental en combinatoria, probabilidad y muchas áreas de las matemáticas.
  • 2.1.4 Permutaciones: Las permutaciones son arreglos ordenados de un conjunto de elementos. En una permutación, el orden de los elementos es importante. Las permutaciones se utilizan para contar y organizar elementos en diferentes órdenes posibles.
  • 2.1.5 Combinaciones: Las combinaciones son arreglos de un conjunto de elementos en los que el orden no importa. Es decir, las combinaciones cuentan cuántas formas distintas hay de seleccionar un subconjunto de elementos de un conjunto más grande, sin considerar el orden de los elementos seleccionados.
  • 2.1.6 Diagrama de Árbol: Un diagrama de árbol es una representación gráfica utilizada para visualizar y analizar las múltiples opciones y resultados posibles de un proceso que implica decisiones secuenciales. Cada nodo del árbol representa un punto de decisión o evento, mientras que las ramas conectan estos nodos para mostrar las diferentes elecciones disponibles en cada paso.
  • 2.1.7 Teorema del Binomio: El teorema binomial nos dice cómo desarrollar expresiones de la forma (a+b)ⁿ, por ejemplo (x+y)⁷. Mientras mayor sea la potencia, es más difícil desarrollar expresiones como esta directamente.

    2.2 Teoría elemental de probabilidad:

  • La Teoría Elemental de Probabilidad es un campo fundamental de las matemáticas que estudia la cuantificación y el análisis de eventos aleatorios. Se centra en calcular y entender las probabilidades de ocurrencia de diversos resultados en situaciones inciertas o aleatorias. Utiliza modelos y principios para determinar cómo se distribuyen los posibles resultados de un experimento aleatorio y cómo calcular la probabilidad asociada con eventos específicos dentro de ese contexto.

    2.3 Probabilidad de Eventos:

  • Espacio muestral: Se refiere al conjunto de todos los posibles resultados individuales de un experimento aleatorio. Es decir, representa el conjunto completo de todos los resultados posibles que podrían ocurrir en una situación particular bajo condiciones especificadas.
  • Evento: n evento es un subconjunto del espacio muestral S, que representa un conjunto de resultados posibles de un experimento aleatorio.
  • Simbología: Se refiere al uso y estudio de símbolos, es decir, signos gráficos que representan conceptos, ideas, operaciones matemáticas, y más.
  • Unión: En matemáticas y teoría de conjuntos, la "unión" de dos conjuntos A y B, denotada como A∪B, es otro conjunto que contiene todos los elementos que pertenecen a al menos uno de los conjuntos originales.
  • Intersección: La intersección de dos conjuntos A y B, denotada como A∩B A∩B, es el conjunto que contiene todos los elementos que son comunes a ambos conjuntos originales A y B.
  • Diagramas de Venn: Los diagramas de Venn son representaciones visuales que utilizan círculos o elipses para ilustrar las relaciones entre conjuntos y sus elementos. Cada conjunto se representa mediante un círculo o una región cerrada, y las áreas de superposición entre estos círculos muestran los elementos que pertenecen simultáneamente a los conjuntos correspondientes.

    2.4 Probabilidad con Técnicas de Conteo:

  • Axiomas: Los axiomas son principios fundamentales aceptados como verdaderos sin necesidad de demostración dentro de un sistema formal o teoría. Sirven como la base sobre la cual se construyen teoremas y se desarrolla el razonamiento dentro de ese sistema, proporcionando consistencia, universalidad y independencia.
  • Teoremas: Un teorema es una proposición matemática que ha sido demostrada como verdadera a partir de axiomas, definiciones previamente establecidas y otros teoremas ya probados.

    2.5 Probabilidad condicional:

  • Dependiente: Se refiere a la situación en la cual la ocurrencia del evento B afecta la probabilidad de que ocurra el evento A.
  • Independiente: Se da cuando la ocurrencia del evento B no tiene ningún efecto sobre la probabilidad de que ocurra el evento A.

    2.6 Ley multiplicativa:

  • se refiere a una regla fundamental que permite calcular la probabilidad de la intersección de dos eventos, bajo ciertas condiciones. Si A y B son dos eventos en un espacio muestral finito o infinito, entonces la probabilidad de que ambos eventos ocurran simultáneamente (es decir, la intersección de A y B) se puede calcular como el producto de la probabilidad de uno de los eventos por la probabilidad condicional del otro evento dado el primero.

    2.7 Eventos independientes:

  • Regla de Bayes: La intersección de dos conjuntos A y B, denotada como A∩B A∩B, es el conjunto que contiene todos los elementos que son comunes a ambos conjuntos originales A y B. La regla de Bayes, también conocida como teorema de Bayes, es una herramienta fundamental en la teoría de la probabilidad que permite actualizar la probabilidad de una hipótesis H dado un nuevo conocimiento o evidencia E.

    3.1 Variables aleatorias discretas:

  • Las variables aleatorias discretas son variables aleatorias cuyo intervalo intervalo de valores valores es finito o contablemente infinito.
  • Distribución de probabilidad en forma general: Una distribución de probabilidad es aquella que permite establecer toda la gama de resultados probables de ocurrir en un experimento determinado. Es decir, describe la probabilidad de que un evento se realice en el futuro.
  • Valor esperado: Cuando la variable aleatoria es discreta, la esperanza es igual a la suma de la probabilidad de cada posible suceso aleatorio multiplicado por el valor de dicho suceso.
  • Variancia: Es un promedio ponderado de las desviaciones al cuadrado de una variable aleatoria de su media. Los pesos son las probabilidades.
  • desviación estándar: Es una medida de dispersión que se emplea para indicar que tan cercanos de la media se encuentran los elementos de la colección
  • Función acumulada: Es una función que describe la probabilidad acumulada de que la variable aleatoria tome un valor menor o igual que un valor específico.

    3.2 Variables aleatorias Continuas:

  • Se dice que una variable aleatoria X es continua si su conjunto de posibles valores es todo un intervalo (finito o infinito) de números reales.
  • Distribución de probabilidad en forma general: Puede tomar cualquier valor num´erico dentro de un intervalo, de modo que entre cualesquiera dos de ellos siempre existe otro posible valor.
  • Valor esperado: Es una medida central que indica el promedio ponderado de todos los posibles valores que la variable puede tomar, considerando sus respectivas probabilidades.
  • Variancia: La varianza cuantifica cuánto se desvían los valores de la variable respecto a su valor esperado (media).
  • Desviación estándar: Es una medida de la dispersión de los valores que puede tomar la variable, indicando qué tan dispersos están estos valores respecto a su media (valor esperado).
  • Función acumulada: Es una función que describe la probabilidad acumulada de que la variable aleatoria tome un valor menor o igual que un valor específico.
  • Cálculos de probabilidad: Los cálculos de probabilidad se refieren a la determinación de la probabilidad de que la variable aleatoria tome valores dentro de un intervalo específico.

Formulas de distribuciones de Probabilidad

Distribución binomial:

$$ f(k; n, p) = \binom{n}{k} p^k (1-p)^{n-k} $$

¿Cúando usar?

  • Experimentos de Bernoulli con dos resultados posibles.

    • Distribución hipergeométrica:

    $$ f(k; N, n, K) = \frac{\binom{K}{k} \binom{N-K}{n-k}}{\binom{N}{n}} $$

    ¿Cúando usar?

  • Muestreo sin reemplazo de una población finita.

    • Distribución de Poisson:

    $$ f(k; \lambda) = \frac{e^{-\lambda} \lambda^k}{k!} $$

    ¿Cuándo usar?

  • Modelar eventos raros y discretos en un intervalo de tiempo o espacio.

    • Distribución Normal:

    $$ f(x; \mu, \sigma^2) = \frac{1}{\sqrt{2 \pi \sigma^2}} e^{-\frac{(x - \mu)^2}{2 \sigma^2}} $$

    ¿Cuándo usar?

  • Modelar datos continuos con una distribución simétrica.

    Regresión lineal

      Es una técnica de análisis de datos que predice el valor de datos desconocidos mediante el uso de otro valor de datos relacionado y conocido.

      5.1 Regresión y correlación:

    • La regresión se usa para modelar y analizar la relación entre una variable dependiente y una o más variables independientes. Su objetivo es predecir o estimar el valor de la variable dependiente en función de las variables independientes.
    • La correlación indica la fuerza y la dirección de la relación lineal entre dos variables. Se mide mediante un coeficiente de correlación, cuyo valor varía entre -1 y 1. Un valor de 1 indica una correlación perfectamente positiva, -1 indica una correlación perfectamente negativa y 0 indica que no hay una relación lineal.
    • La regresión busca una ecuación para predecir Y a partir de X, mientras que la correlación solo mide la fuerza de la asociación entre las variables.

      5.1.1 Diagrama de dispersión:

    • Es un gráfico donde muestra los pares de datos de dos variables, representados con puntos en un plano cartesiano. Este nos indica patrones de relación entre las variables.

      5.1.2 Regresión lineal simple:

    • Es una técnica estadística que encuentra la mejor línea recta que describe la relación entre una variable independiente y una variable dependiente.

      5.1.3 Correlación:

    • Es la medida estadística que indica la fuerza y la dirección de la relación lineal entre dos variables, con un valor que varía entre -1 y 1.

      5.1.4 Determinación y análisis de los coeficientes de correlación y de determinación:

    • Coeficiente de correlación (r): Mide la fuerza y dirección de una relación lineal entre dos variables. Su valor está entre -1 y 1.
    • Coeficiente de determinación (R²): Indica el porcentaje de la variación en la variable dependiente que puede ser explicada por la variable independiente. Su valor está entre 0 y 1.

      5.1.5 Distribución normal bidimensional:

    • Representación de la distribución conjunta de dos variables continuas, formando una superficie en forma de campana que describe la densidad de probabilidad conjunta.

      5.1.6 Intervalos de confianza y pruebas para el coeficiente de correlación:

    • Intervalos de confianza: Rango dentro del cual es probable que se encuentre el valor real de la correlación en la población, con un cierto nivel de confianza.
    • Pruebas de hipótesis: Evaluaciones que determinan si la correlación observada en una muestra es significativamente diferente de cero, indicando una relación real en la población.

      5.1.7 Errores de medición:

    • Diferencias entre el valor medido de una variable y su valor real, debidas a instrumentos de medición imprecisos, errores humanos o sesgos del observador.

      6.1 Muestreo:

    • Esta técnica estadística sirve para obtener informacion acerca de una población a partir del estudio de una submuestra de la misma.

      6.1.1 Tipos de muestreo:

    • Se encuentran divididos en dos categorías de muestreo en estadística y son: probabilístico y no probabilístico.

      • Muestreo no probabilístico: Es una técnica de muestreo donde las muestras se recogen por medio de un proceso que no les brinda a todos los individuos de la población las mismas oportunidades de ser seleccionados.

    • Muestreo por conveniencia: Se seleccionan miembros de la población que están fácilmente disponibles.
    • Muestreo deliberado, crítico o por juicio: Es donde el investigador selecciona intencionalmente individuos considerados representativos o críticos, basándose en su conocimiento y juicio, para proporcionar información relevante sobre el tema de investigación.
    • Muestreo por cuotas: Se seleccionan miembros de la población para cumplir con ciertas cuotas predeterminadas.
    • Muestreo bola de nieve: Los participantes iniciales reclutan a otros participantes de entre sus conocidos.

      • Muestreo probabilístico: Es una técnica en la cual las muestras son recogidas mediante un proceso que le brinda a todos los individuos de la población la misma oportunidad de ser seleccionados.

    • Muestreo aleatorio simple: Es el método de muestreo básico utilizado en métodos estadísticos y cálculos. Para recopilar una muestra aleatoria simple, a cada unidad de la población objetivo se le asigna un número. Luego se genera un conjunto de números aleatorios y las unidades que tienen esos números son incluidas en la muestra.
    • Muestreo sistemático: Es aquel en el que los elementos de la población se ponen en una lista y luego cada enésimo elemento de la lista se selecciona sistemáticamente para su inclusión en la muestra.
    • Muestreo estratificado: Se utiliza cuando el investigador quiere resaltar subgrupos específicos dentro de una población.
    • Muestreo por conglomerados: Puede ser utilizado cuando es imposible o impráctico elaborar una lista exhaustiva de los elementos que constituyen a la población objetivo

      6.1.2 Teorema de Límite central:

    • Establece que, bajo ciertas condiciones, la suma de un gran número de variables aleatorias independientes e idénticamente distribuidas, tiende a distribuirse normalmente, sin importar la distribución original de las variables.

      6.1.3 Distribución muestral de la media:

    • Representa cómo se distribuirían las medias de múltiples muestras tomadas de la misma población.

      6.1.4 Distribución muestral de una proporción:

    • Describe cómo se distribuyen las diferentes proporciones que se podrían obtener al extraer repetidamente muestras aleatorias del mismo tamaño de la misma población.

      6.2 Estimación:

    • Se refiere al proceso de aproximar el valor de un parámetro desconocido de una población a partir de los datos de una muestra.

      6.2.1 Estimación puntual:

    • Es el proceso de obtener un único valor que se considera la mejor aproximación del valor verdadero de un parámetro desconocido de una población.

      6.2.2 Estimación por intervalo:

    • Se utiliza para estimar un rango de valores que contiene el verdadero valor de un parámetro poblacional con un nivel de confianza específico.

      6.2.3 Intervalo de confianza para una media:

    • Es un rango de valores que contiene el verdadero valor de la media poblacional con un nivel de confianza específico. Se utiliza para estimar la media de una población a partir de los datos de una muestra.

      6.2.4 Intervalo de confianza para una proporción:

    • Se utiliza para estimar la proporción de una característica específica en una población a partir de los datos de una muestra.

      6.3 Prueba de hipótesis:

    • Se utiliza para determinar si existe evidencia suficiente para rechazar una afirmación preexistente sobre un parámetro poblacional, conocida como hipótesis nula.

      6.3.1 Errores tipo I y II:

    • Error tipo I: Se comete un error de tipo I cuando se rechaza la hipótesis nula (H0) siendo esta verdadera. Dicho de otra forma se concluye que existe una diferencia o efecto cuando en realidad no la hay.
    • Error tipo II: Se comete un error de tipo II cuando no se rechaza la hipótesis nula (H0) siendo esta falsa. En otras palabras, se concluye que no existe una diferencia o efecto cuando en realidad sí la hay.

      6.3.2 Pasos para realizar una Prueba de Hipótesis:

    • 1.- Plantear la hipótesis.
    • 2.- Establecer un nivel de significancia.
    • 3.- Seleccionar el estadístico de prueba a aplicar.
    • 4.- Establecer una regla de decisión.
    • 5.- Tomar una decisión o conclusión.

      6.3.3 Prueba de hipótesis para una media:

    • Es un procedimiento estadístico formal que permite determinar si la media de una población es igual a un valor específico (hipótesis nula), con base en la evidencia proporcionada por una muestra aleatoria de dicha población.

      6.3.4 Prueba de hipótesis para una proporción:

    • Permite evaluar si una característica o evento específico ocurre con una frecuencia particular en una población.
